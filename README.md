# Mock Generator

Gerador de dados mock em python

> Projeto não terminado

## Run

1. Crie o venv
```
python -m venv venv
```

2. Ative o venv
    
```
venv\Scripts\activate
```

3. Instale as dependências

```
pip install -r requirements.txt
```

4. Inicie a aplicação

```
uvicorn main:app --reload
```
